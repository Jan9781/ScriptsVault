Getting Started

Welcome to ScriptVault. Our repository provides high-quality scripts for various use cases.

How to use Tampermonkey scripts:

  1. Install the Tampermonkey extension for your browser.
  2. Click the "Copy Code" button on any script card.
  3. Open the Tampermonkey dashboard and create a new script.
  4. Paste the code and save.
